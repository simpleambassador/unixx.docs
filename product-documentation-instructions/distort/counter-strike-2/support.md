## Common Issues

1. Intel Driver Error
   * Close all antiviruses
   * Close all anticheats (e.g., FACEIT, Akros)
   * Close all debuggers
   * Disable Core Isolation in Windows Security
   * Restart your PC
2. Connection Error
   * Connect to a VPN (e.g., ProtonVPN) before running the loader
   * You can disable the VPN after injection
3. Empty Fields
   * Make sure your username and password are not blank
   * If switching accounts, re-open the loader<br>
