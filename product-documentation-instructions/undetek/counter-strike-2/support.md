## Common Issues

1. 0x1 Error
   * Restart your PC.
   * Right-click the Steam icon and select **Run as administrator**.
   * Remove Steam from beta mode.
   * Remove CS2 from beta mode.
   * Open CS2 to the main menu.
   * Press **SHIFT + TAB** and confirm the Steam overlay opens.
   * Right-click the cheat file and run it as administrator.
2. Disabling FACEIT Anti-Cheat and Kernel Anti-Cheats
   * Find the FACEIT Anti-Cheat icon near your system clock (bottom-right system tray).
   * Right-click the icon and select **Disable** (may require admin permissions).
   * Repeat for any other kernel-level anti-cheat programs you have.
3. Enabling Steam Overlay in Steam Settings
   * Open Steam, click **Steam** in the top-left, then select **Settings**.
   * Go to the **In-Game** tab.
   * Check **Enable the Steam Overlay while in-game**.
   * Click **OK** to save.
